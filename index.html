Comment fonctionne l'inscription

Remplissez le formulaire, puis choisissez de payer avec MonCash ou avec UNIBANKOnline. Simple, sans frais cachés.

Étape
01

Cliquez sur «S'inscrire» pour un module

Choisissez le module que vous voulez et cliquez sur le bouton «S'inscrire» pour ouvrir le formulaire d'inscription.

Étape
02

Remplissez le formulaire & choisissez votre moyen de paiement

Indiquez votre nom, prénom, téléphone et email. Nous recevons vos informations, puis vous êtes redirigé vers MonCash ou vers UNIBANKOnline selon votre choix.

Étape
03

Complétez le paiement

Terminez le paiement dans l'application MonCash ou sur le portail UNIBANKOnline. Vous recevez l'accès en moins de 24h après confirmation.

Important : comme le paiement se fait sur un compte personnel, la vérification est faite manuellement. Votre place dans la formation n'est confirmée qu'après réception & vérification du paiement — ne supprimez donc aucun message de confirmation (MonCash ou UNIBANKOnline) tant que vous n'avez pas reçu votre accès. Pour toute question, contactez-nous sur WhatsApp.

Les 4 modules sont indépendants : vous pouvez suivre celui de votre choix, dans l'ordre qui vous convient. À l'intérieur d'un même module, les leçons se débloquent dans l'ordre — impossible de passer à la leçon suivante avant d'avoir terminé la précédente. Dès qu'un module est complété à 100%, son certificat se débloque immédiatement.
